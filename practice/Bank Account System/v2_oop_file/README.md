# Bank Account System (OOP + File I/O) 🏢

A console-based Bank Account System built using Python.  
This version uses **Object-Oriented Programming** and **file handling** to persist data.

## Features
- Add records/tasks
- View stored records
- Data is saved to a file
- Data persists even after program restart
- Menu-driven CLI interface

## Concepts Used
- Object-Oriented Programming (Classes & Methods)
- File Handling (read/write)
- Exception Handling
- Lists and basic control flow

## Project Structure

├── Main.py
└── bank.txt
