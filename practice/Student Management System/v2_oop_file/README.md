## 📌 Overview
A simple Student Management System created using Python OOP.  
The program allows storing student data in txt formate, adding marks, and evaluating performance.

---

## 🧠 Concepts Used
- Classes & Objects
- Lists
- Methods
- Average Calculation
- Pass / Fail Logic

---

## 🚀 Features
- Add Student Marks
- Calculate Average
- Pass / Fail Evaluation
- Display Student Information
- Save Information into student.txt

---

## 🛠 Tech Stack
- Python 3
- Object Oriented Programming
- I/O Management
