# 📚 Library Management System (Python)

A simple **command-line based Library Management System** written in Python.  
It allows users to add books, view available books, issue books, and return books.  
All data is stored persistently using a text file.

---

## 🚀 Features

- Add new books to the library
- View all books with their status (Available / Issued)
- Issue a book by title
- Return a book by title
- Data is saved in a file (`library.txt`) / (`library.json`)
- Simple menu-driven interface

---

## 🛠️ Technologies Used

- Python 3
- File Handling (Text File)
- Object-Oriented Programming (OOP)
