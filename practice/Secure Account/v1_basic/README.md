# 🔐 Secure Account System (Python OOP)

**Author:** Dhruv  

A Python project designed to demonstrate **Object-Oriented Programming (OOP)** concepts by modeling a secure account system.

This project focuses on class design, object behavior, and data encapsulation without emphasizing user interaction or input/output handling.

---

## 📌 Project Overview

The Secure Account System represents a simplified account model where core banking-related operations are implemented using object-oriented principles.

The project highlights how data and behavior can be bundled together inside a class while maintaining controlled access to sensitive information.

---

## 🧠 OOP Concepts Used

- Encapsulation  
- Classes and Objects  
- Instance Attributes  
- Methods  
- Data protection through controlled access  

---

## 🧩 System Design

The system is structured around a single class that:

- Stores account-related information  
- Maintains internal state such as balance  
- Provides methods to modify and access data securely  
- Restricts sensitive operations using logical checks  

All operations are handled through class methods, ensuring proper separation of data and behavior.
